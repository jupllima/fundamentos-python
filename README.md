# Códigos

|comando|descrição          |

|ls     | listar            |

```
ls: listar
mkdir: criar diretório 
cd: voltar pra raíz
pwd: mostrar o diretório
cd: volta as pastas
clear: limpar
Ctrl + X + S: salvar as alterações
history: ver as modificações feitas anteriormente
```
